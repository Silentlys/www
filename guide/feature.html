# 短信

注意！

当您刷入 Open Gapps Micro 或更完整的 Open Gapps，内置短信程序会替换为 Google 版本，以下特色功能将不再有效。

魔趣建议需要使用 Open Gapps 的用户刷入 Open Gapps Nano 以确保功能完整性。

# 号码归属地 MK44.4+

短信列表中会显示发送方的号码归属地。

# 智能归档短信 当前支持中国大陆、中国香港、马来西亚与印度号码的识别 MK71.2+

当下，我们的短信程序中充斥着各类通知与验证码短信。它们的存在使得我们检索与好友间的日常短信变得极为困难。因此，魔趣将帮助您自动归档此类短信，让重要短信变得清晰可见。

# 自动识别验证码短信 MK60.1+

魔趣会自动识别验证码类短信，您只需点击通知即可一键复制验证码，提高效率。

# 自动删除验证码短信 MK71.2+

验证码短信具有时效性，长期保存是没有意义的。开启该选项，当您复制验证码后，短信会自动销毁。

# 归档短信未读角标 MK71.2+

魔趣为归档短信加入了未读角标，避免你错过需要及时阅读的短信。

# 可左右滑动的归档界面 MK60.1+

Google 并未在 Android 开源项目 (AOSP) 中添加短信归档界面的滑动支持，魔趣完善了它。

# 短信快捷处理选项 MK60.1+

收到新通知时，Android 原生版本 只有一个"回复"选项，魔趣增加了"拨打"和"已读"选项。

# 电话

注意！

当您刷入 Open Gapps Micro 或更完整的 Open Gapps，内置电话程序会替换为 Google 版本，以下特色功能将不再有效。

魔趣建议需要使用 Open Gapps 的用户刷入 Open Gapps Nano 以确保功能完整性。

# T9 拨号 当前支持汉语、韩语、希腊语、俄语、希伯来语和乌克兰语 MK44.4+

您可以在拨号面板通过输入号码、姓名全拼或是首字母的方式来检索联系人

# 号码归属地 MK44.4+

会在以下位置会显示号码归属地：

  • 拨号面板
  • 通话记录
  • 联系人详情
  • 来电、去电
  • 来电浮动通知
  • 未接来电通知

# 中文姓氏联系人头像

Android 原生会提取英文联系人的首字母作为头像，魔趣加入了中文姓氏的支持。
